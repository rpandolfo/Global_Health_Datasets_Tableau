# Tableau Global Health Dashboard Project

This project presents an interactive Tableau dashboard created during a **Data Technician Bootcamp**, focusing on **global health data analysis**. The dashboard visualizes trends and distributions related to liver cancer, life expectancy, and BMI across countries and continents.

## Key Features

- **Interactive Dashboards:** Explore global health statistics with insights on liver cancer prevalence, life expectancy, and distribution across the globe.  
- **Filters & Slicers:** Dynamically filter data by continent, country, and gender.  
- **Variety of Charts:**
  - **Maps:** Display the global distribution of liver cancer cases.  
  - **Pie Charts:** Show liver cancer distribution by gender.  
  - **Bar Charts:** Show life expectancy by continent.  
  - **Scatter Plots:** Visualize life expectancy versus BMI across the five continents.  
  - **Line Charts:** Track life expectancy trends over time for the top 5 countries.  
- **Data Storytelling:** The dashboard is structured to guide the audience through key global health insights and trends.
